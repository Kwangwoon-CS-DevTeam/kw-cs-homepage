## 🛠️ 기술 스택
| **Category** | **Stack** |
|:------------:|:----------:|
| Back-end | SpringBoot |
| Database | MySQL |
| Front-end | React |

## 💻 설치 방법
   1. 의존성을 설치합니다:
      ```bash
      npm install
      ./gradlew build
      ```
   2. 애플리케이션을 시작합니다:
      ```bash
      npm start
      ./gradlew bootRun
      ```
