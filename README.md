# 🩵 프로젝트 제목
컴퓨터정보공학부 홈페이지

## 🚀 소개
컴퓨터정보공학부의 공지사항, 자료, 정보, QnA 등이 있는 홈페이지입니다.

## 🛠️ 기술 스택
| **Category** | **Stack** |
|:------------:|:----------:|
| Back-end | NodeJS |
| Database | Supabase |
| Front-end | React |

## 💻 설치 방법
   1. 의존성을 설치합니다:
      ```bash
      npm install
      ```
   2. 애플리케이션을 시작합니다:
      ```bash
      npm start
      ```
